# 🐉 Wyvern Agent

> *超越爬行，振翅翱翔的超级智能体*

Wyvern Agent 是一个开源的自主智能体框架，为**真正的超级智能**而生：

- 🧠 **超记忆** — 超越 RAG，具备反思和联想召回能力
- 🔧 **无限工具** — 自动发现和编排任意工具
- 🧬 **自我进化** — 智能体自己写代码、测代码、改进代码
- 🌍 **跨平台** — 桌面、网页、移动端、命令行 — 一处编写，处处运行
- 🕸️ **多智能体集群** — 自动分解任务，并行执行
